Strategies for avoiding layout shift with a progressively enhanced burger menu (demos)

A series of demos showing strategies for a progressively enhanced "burger" menu without sacrificing an excellent Cumulative Layout Shift metric score.

The demos that use JavaScript all use the same external dependency. The Sky Nav “init” function has a 1-second delay to exaggerate the length of time you see the layout shift, mimicking a slow-loading JavaScript file.

Read Cloud Four's "Strategies for avoiding layout shift with a progressively enhanced burger menu" article for demo details.
